// Décorateurs class-validator avec leurs options
/* 

Je vais enrichir les informations en ajoutant les options disponibles pour chaque décorateur de validation.

J'ai créé un fichier TypeScript complet qui détaille chaque décorateur de validation de class-validator avec toutes les options qu'ils acceptent.

Voici comment les informations sont organisées :

1. Chaque catégorie de validateurs est dans un tableau séparé, comme :
   - `"string"ValidatorsWithOptions` (validateurs de chaînes)
   - `"number"ValidatorsWithOptions` (validateurs numériques)
   - `dateValidatorsWithOptions` (validateurs de dates)
   - `arrayValidatorsWithOptions` (validateurs de tableaux)
   - etc.

2. Chaque entrée du tableau contient :
   - `decorator` : le nom du décorateur (exemple : `@IsEmail`)
   - `options` : un objet détaillant toutes les options spécifiques que ce décorateur accepte

3. J'ai également inclus un objet `commonOptions` à la fin qui liste les options communes qui s'appliquent à presque tous les décorateurs comme :
   - `each` : pour appliquer le validateur à chaque élément d'un tableau
   - `message` : pour spécifier un message d'erreur personnalisé
   - `groups` : pour définir des groupes de validation
   - `always` : pour toujours valider même si la propriété n'est pas définie
   - `context` : pour fournir un contexte de validation

Ces informations peuvent être utilisées comme référence complète lors de l'implémentation de la validation avec class-validator dans votre projet TypeScript.
*/
