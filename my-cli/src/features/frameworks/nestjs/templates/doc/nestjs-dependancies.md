Dépendances de production (prod) :
@nestjs/swagger → Génération de documentation OpenAPI (Swagger).

```bash
npm install @nestjs/swagger
```

@nestjs/typeorm → ORM TypeORM pour la gestion des bases de données.

```bash
npm install @nestjs/typeorm
```

@nestjs/config → Gestion des variables d’environnement et configurations.

```bash
npm install @nestjs/config
```

@nestjs/mapped-types → Utilitaire pour transformer les DTOs.

```bash
npm install @nestjs/mapped-types
```

class-transformer → Transformation des objets en TypeScript.

```bash
npm install class-transformer
```

class-validator → Validation des DTOs.

```bash
npm install class-validator
```

ts-node →

```bash
npm install ts-node
```
