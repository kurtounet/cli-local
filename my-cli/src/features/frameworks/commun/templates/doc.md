Voici l’inventaire complet et actuel des **types d’`<input>` HTML** reconnus en HTML5, classés par familles fonctionnelles. C’est exhaustif.

---

### 1. **Champs textuels**

```html
<input type="text">
<input type="password">
<input type="email">
<input type="search">
<input type="tel">
<input type="url">
<input type="number">
<input type="hidden">
```

---

### 5. **Fichiers et médias**


---

### 6. **Couleur**

```html
<input type="color">
```

---

### 7. **Champs de sécurité / auto-complétion**

```html
<input type="autocomplete">   <!-- pseudo notation, l'attribut autocomplete s'applique à tous les <input> -->
```

Ce n’est pas un type, mais beaucoup se trompent :
`autocomplete="off"`/`on` est un **attribut** et non un input.
